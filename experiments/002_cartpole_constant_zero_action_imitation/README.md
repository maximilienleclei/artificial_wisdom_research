# 002_cartpole_constant_zero_action_imitation

Static MLP genetic algorithm imitating a trivial CartPole policy that always predicts action zero.

Run from this folder after installing `code/` into the active venv:

```powershell
C:\Users\Max\venv\Scripts\python.exe -m awr.experiments.act_pred_cartpole --generations 40 --population-size 64
```
